## üìù Key Concepts Learned

### 1. **Images (`<img>`)**

- **Source (`src`)**: Linking to files (local or URL).
- **Alt Text (`alt`)**: Providing descriptions for accessibility and SEO.
- **Width & Height**: Controlling image size.
- **Object Fit**: Fitting images into containers properly (e.g., `object-fit: cover`).

### 2. **Form Inputs (`<input>`)**

- **Types**: `text`, `search`, `email`, `password`.
- **Placeholders**: Hint text inside empty inputs.

### 3. **Styling Forms**

- Adding padding for comfort.
- Rounding corners (`border-radius`).
- Adding a focus state (`:focus`) to highlight active inputs.
- Styling the placeholder text (`::placeholder`).
