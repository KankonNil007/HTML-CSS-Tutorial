### 1. **The CSS Box Model**

Every element on a web page is a rectangular box comprised of:

- **Content**: The actual text or image.
- **Padding**: Space _inside_ the border, around the content.
- **Border**: A line wrapping the padding and content.
- **Margin**: Space _outside_ the border, separating the element from neighbors.

### 2. **Chrome Developer Tools**

- Inspecting elements (Right-click -> Inspect).
- Toggling styles on/off to debug.
- Viewing the computed box model diagrams.

### 3. **Spacing Techniques**

- Removing default browser margins (`margin: 0` on body).
- Using margins to separate elements.
