One of the most important concepts in CSS. The `display` property controls how an element behaves in the document flow.

## ğŸ“ Key Concepts Learned

### 1. **Block vs. Inline**

- **Block (`display: block`)**: Takes up the full width, starts on a new line (e.g., `<div>`, `<h1>`, `<p>`).
- **Inline (`display: inline`)**: Takes only necessary width, flows with text (e.g., `<span>`, `<a>`, `<img>`).

### 2. **Inline-Block**

- **`display: inline-block`**: The best of both worlds. Allows setting width/height/padding like a block, but sits side-by-side like inline.
- Crucial for layouts before Flexbox/Grid existed.

### 3. **None**

- **`display: none`**: Removing the element from the document flow entirely (not just invisible, but gone).
