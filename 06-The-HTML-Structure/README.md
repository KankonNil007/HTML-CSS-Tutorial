### 1. **HTML Boilerplate**

- `<!DOCTYPE html>`: Telling the browser this is an HTML5 document.
- `<html>`: The root element wrapping everything.
- `<head>`: Container for metadata, title, and links (not visible on page).
- `<body>`: Container for visible content.

### 2. **Metatags**

- `<meta charset="UTF-8">`: Ensuring proper text encoding.
- `<meta name="viewport" ...>`: Ensuring the page scales correctly on mobile devices.

### 3. **Linking Resources**

- **Title**: `<title>` sets the text in the browser tab.
- **CSS**: `<link rel="stylesheet" href="...">` connects external styles.
- **JavaScript**: `<script src="...">` connects interactivity.
