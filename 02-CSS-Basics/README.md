# üé® CSS Basics

This module introduces **Cascading Style Sheets (CSS)**, the language responsible for the visual presentation of web pages.

## üìù Key Concepts Learned

### 1. **Selectors**

- Targeting elements by their tag name (e.g., `button`, `p`).
- Understanding how CSS rules apply to HTML elements.

### 2. **Colors & Backgrounds**

- Setting text color using the `color` property.
- Changing backgrounds with `background-color`.
- Using named colors (e.g., `red`), hex codes (`#ff0000`), and RGB values.

### 3. **Fundamental Properties**

- **Typography**: Modifying text size with `font-size`.
- **Borders**: Adding outlines using `border`, `border-width`, `border-style`, and `border-color`.
- **Rounding**: create rounded corners with `border-radius`.

### 4. **Methods of Styling**

- **Inline Styles**: Using the `style` attribute directly on elements.
- **Internal CSS**: Writing styles within the `<style>` tag in the `<head>`.
