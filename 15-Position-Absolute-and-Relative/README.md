# üéØ Position Absolute & Relative

This module dives deep into the powerful relationship between `position: absolute` and `position: relative`.

## üìù Key Concepts Learned

### 1. **Relative Positioning**

- `position: relative;`: The anchor point. It establishes a positioning context for children without removing itself from the document flow.

### 2. **Absolute Positioning**

- `position: absolute;`: Removing elements from the flow and placing them precisely within the nearest **positioned** ancestor.
- Using `top`, `bottom`, `left`, `right` coordinates.

### 3. **Common Use Cases**

- **Overlays**: Placing text or badges over images (e.g., video time duration).
- **Tooltips**: Creating custom popups that appear relative to buttons.
- **Close Buttons**: Positioning "X" buttons in the top-right corner of a modal.

## üìÇ Practical Examples

- **Video Time Badge**: Positioning the timestamp in the corner of a thumbnail.
- **Notification Counts**: Adding a red number badge on a bell icon.
